### Usage

```
usage: qrbf.py [-h] -d FILE [-p PORT] [-i MILLISECONDS]

optional arguments:
  -h, --help       show this help message and exit
  -d FILE          dictionary FILE
  -p PORT          server port (8080 default)
  -i MILLISECONDS  brute force interval (1000 ms default)
```
